## 🚀 Project Overview

The **Library Equipment Management System** is a digital solution designed to manage and monitor equipment available in libraries or institutional labs.  

The system allows administrators to track equipment, organize departments and libraries, and monitor the working status of devices.

This application replaces manual record keeping with a **structured and efficient digital system**.

---

## 🎯 Problem Statement

Many institutions still manage library equipment manually which leads to:

- Difficulty tracking equipment
- Inefficient resource management
- Lack of real-time information about equipment status

This project provides a **centralized digital system** to manage equipment effectively.

---

## 🧠 Key Features

- Add and manage **Departments**
- Add **Libraries within departments**
- Grid-based interface for organizing equipment
- Track **working and non-working equipment**
- Store equipment information
- Generate statistics about equipment availability
- Easy navigation between departments and libraries

---

## 📊 System Capabilities

The system provides statistics such as:

- Total number of equipment
- Number of working equipment
- Number of non-working equipment
- Percentage of working devices

These insights help administrators quickly identify issues.
